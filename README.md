# App-Predictive-Maintenance
Sistema de previsão de falhas industriais
Projeto desenvolvido para monitoramento Predidtivo de mancais de esteriras transportadores Utilizando:
- Python
- Machine Learning
- Streamlit
- Análise de Dados.

O Sistema analisa:
- Temperatura
- Vibração
- Corrente do motor
- Tensão da correia

e calcule:
- Healt Score
- Probabilidade de Falha
- Nível de Risco
- Recomendação de Intervenção
